Making easy to run Docker images

Liferay Portal Nook, a fast and flexible CLI for managing Liferay Portal's Docker images.

Run Liferay Portal instances

Don't worry anymore about having to install build tools to start your Liferay Portal instance. Start the Liferay Portal released version you prefer, including specific Liferay products like "Commerce", using containers.

Configure your instance

Do you need a special configuration like a database configuration, or an LDAP integration? Specify the location of your custom portal properties file to configure your instance.
