Discover lpn

Making easy to run Docker images

Liferay Portal Nook (lpn), a fast and flexible CLI for managing Liferay Portal's Docker images.

Run Liferay Portal instances

Don't worry anymore about having to install build tools to start your Liferay Portal instance. Start the Liferay Portal released version you prefer, using Docker containers.

 Read More

Configure your instance

Do you need a special configuration like a database configuration, or an LDAP integration? Specify the location of your custom portal properties file to configure your instance.

 Read More

Deploy apps to your portal

Imagine a developer gives you a non-released JAR/WAR file to test it. You could deploy and test it in seconds! And everything with the power of one single command.
